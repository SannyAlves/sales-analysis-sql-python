## Tecnologias
- MySQL
- SQL
- Python (Pandas, Matplotlib)

## Descrição
Projeto de análise de dados de vendas utilizando SQL para extração
e Python para análise e visualização.

## Análises realizadas
- Faturamento por cidade
- Faturamento por categoria
- Quantidade vendida por produto

## Estrutura
- sql/: queries SQL
- python/: scripts de análise
- images/: gráficos gerados
SELECT * FROM sales;
